Longest sequences of bytes

  1. Input the name of file
  2. Find byte/bytes with the maximum number of repetitions
  3. Output them separated by a space

Examples

All bytes: [ 115, 100, 100, 102, 103, 103, 103, 104, 115, 115]
Result: 103 115

All bytes: [108, 107, 106, 104, 111, 103, 101, 114, 116, 121, 117, 105, 111]
Result: 111

All bytes: [108, 107, 106, 104, 103, 101, 114, 116, 121, 117, 105, 111]
Result: None
